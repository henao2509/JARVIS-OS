# SKILLS

Las habilidades son las capacidades de JARVIS.

Cada habilidad será independiente.

Ejemplos:

- Crear aplicaciones.
- Programar.
- Diseñar páginas web.
- Analizar documentos.
- Leer PDFs.
- Crear imágenes.
- Crear videos.
- Crear automatizaciones.
- Investigar en Internet.
- Gestionar archivos.
- Controlar el computador.
- Enseñar cualquier tema.
- Resolver problemas.
- Crear estrategias de negocio.
- Analizar datos.

Cada habilidad podrá evolucionar sin afectar a las demás.
